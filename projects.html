PROJECTS

Throughout my STEM journey, I have worked on multiple technology-based projects involving Arduino, ESP8266, IoT, Robotics, Sensors, and LEGO systems. Each project helped me gain skills in programming, electronics, creativity, and real-world problem solving. Below is a list of the major projects I have built, explored, or taught.

Temperature-Based Fan Speed Controller

Automatically adjusts fan speed based on temperature readings to improve comfort and save energy.

Virtual Home Assistant (IoT)

Controls home appliances remotely using IoT commands to create a smart home environment.

WiFi Controlled Robot (ESP8266)

A mobile robot that can be controlled using WiFi with obstacle detection support.

Health Monitoring System

Displays heart rate and sensor values using an OLED screen for real-time health monitoring.

Smart Blind Glass (IoT Alert)

A wearable obstacle detection device that helps visually impaired users move safely.

Chatbot (NodeMCU + OLED)

A small embedded chatbot that responds to predefined questions and displays answers on OLED.

Anti-Missile Radar System

Radar simulation using ultrasonic sensors and a servo to detect incoming objects.

Smart Missile Detection System

Detects fast-moving objects using LDR/ultrasonic sensors with alert notifications.

EquitasBOT (NodeMCU Robot)

An interactive classroom robot designed to make learning fun and engaging for students.

Smart Attendance System

Automatically records attendance using sensors and reduces manual work.

e-Voting Machine

A digital voting prototype that securely counts votes and displays instant results.

Robotic Arm

A servo-controlled robotic arm capable of picking and placing objects.

Weather Monitoring System

Monitors temperature and humidity levels and displays them in real time.

Gas Leakage Detection System

Uses a gas sensor to detect leakage and immediately triggers an alert.

Bluetooth Controlled Car

A car that moves using Bluetooth commands sent from a mobile phone.

Arduino Sign Language Trainer

A servo-driven robotic hand that performs sign language gestures for learning.

Animal Robot Helper (LEGO)

A LEGO helper robot that performs simple tasks for learning robotics basics.

LEGO Race Car

A fast-moving LEGO car designed using gears, motors, and mechanical principles.

LEGO Scribbler Robot

A LEGO robot that draws patterns using mounted pens and programmed movement.
